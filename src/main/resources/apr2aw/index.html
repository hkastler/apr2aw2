Weight Measurements

Gradual behavioral changes enhance chances for success.

How to use: enter the starting date, starting weight, and goal weight. Weigh yourself every morning before breakfast. You are to adjust food intake or activity so that your weight falls on the line each day. If you are above the line, eat less and/or move more. If you are below the line, you can eat a little more for that day. Morehouse advocates losing only 1 pound per week, but I believe (from personal experience) that up to 2 pounds per week should also be okay. And if you're not on track to lose 2 pounds per week, just go back to a goal of 1 per week.

You will need a scale with decimal point precision.

Remember: always check with your physician before beginning any weight loss program.

All data is stored locally, so you'll need to use the same browser each time.
